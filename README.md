## 🚀 Cách chạy dự án
### ====== Mới clone về ======
### 1. Cài đặt frontend (React)
#### Lưu ý phải cài Node.js
```bash
cd frontend
npm install
npm start
```
#### 2.cài đặt backend (spring boot)
```bash
cd backend
./mvnw spring-boot:run
```
hoặc
```bash
cd backend
mvnw spring-boot:run
```
